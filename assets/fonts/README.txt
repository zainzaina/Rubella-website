Place font files here to use the exact branding fonts.

Required filenames (or update CSS `@font-face` if different):
- TanPearl-Regular.woff2 (English heading font)
- TanPearl-Regular.woff (optional fallback)
- BaradaReqa-Regular.woff2 (Arabic body font)
- BaradaReqa-Regular.woff (optional fallback)

If you don't have these fonts, you can use close Google Fonts alternatives in `assets/css/style.css`:
- English headings: use `Cormorant Garamond` or `Playfair Display`.
- Arabic body: use `Cairo`, `Tajawal`, or `Noto Naskh Arabic`.

After adding files, refresh the browser to see fonts load.
